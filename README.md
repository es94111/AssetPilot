# VitaTrack 健康管理系統

> **Version 1.0**

個人健康管理應用程式，協助使用者記錄身體組成、飲食攝取、運動與健康指標，透過數據分析做出更好的健康決策。

## 功能模組

| 模組 | 說明 |
|------|------|
| 身體組成紀錄 | 體重、體脂率、肌肉量、身體圍度、趨勢圖表 |
| 飲食紀錄 | 餐次紀錄、食物資料庫、營養素分析、飲水追蹤 |
| 運動紀錄 | 有氧 / 重訓紀錄、消耗熱量估算、訓練量統計 |
| 健康指標 | 血壓、血糖、睡眠、自訂指標追蹤 |
| 目標設定 | 體重 / 營養素 / 運動目標、進度追蹤儀表板 |
| 數據報表 | 每日摘要、週報月報、長期趨勢分析、資料匯出 |

## 技術架構

- **前端**：Web 應用程式（RWD 響應式設計）
- **後端**：RESTful API
- **資料庫**：關聯式資料庫
- **部署**：Docker 容器化
